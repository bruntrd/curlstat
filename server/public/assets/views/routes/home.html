Welcome to CurlStat

Welcome to Curl Stat, a user friendly stat program designed for curlers by a curler. The overall design idea came when seeing coaches from all over the world lugging equipment around while trying to find a place to set up. Laptops are no longer necessary for the app, but can still be used. So explore around a bit on the site and find out more about the app and its use. Either sign up for an account to start your season, or take it for a test run and see how you like the feel.
